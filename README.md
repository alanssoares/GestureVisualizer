# GestureVisualizer
View and apply methods on gestures

## Dependencies

- GCC 4.x

- Python 2.6+/3.x

- Nite v1.5.2.21

- OpenNI v1.5.7.10

- PCL 1.8

## Compile and run (OSX and Linux)

1 - Create the build dir:
** mkdir build
2 - Enter inside the build and run:
** cmake ..
3 - Compile using make
** make
4 - Run:
** ./start -m [option]
